# ToDo List 과제

## 주요 구현 기능
1. **할 일 추가:** 텍스트를 입력하고 '추가' 버튼을 클릭하거나 `Enter` 키를 누르면 리스트에 새로운 항목이 등록됩니다. 빈 값을 넣으면 경고창이 발생합니다.
2. **완료 상태 변경:** 각 항목의 '완료' 버튼을 누르면 글자에 취소선이 적용되며 상태가 변경됩니다.
3. **할 일 삭제:** '삭제' 버튼을 누르면 배열 내 데이터가 삭제되고 화면이 다시 렌더링되어 항목이 즉시 지워집니다.
4. **데이터 영구 보존:** `localStorage`를 활용하여 새로고침을 하거나 브라우저를 닫았다가 다시 열어도 기존 리스트가 파기되지 않고 유지됩니다.

## 활용한 자바스크립트 핵심 기술
과제 구현을 위해 다음과 같은 배운 문법들을 조합하여 사용했습니다.
- **DOM 요소 제어:** `document.querySelector`를 사용하여 HTML 태그를 식별하고 가져왔습니다.
- **화면 업데이트:** `for` 반복문과 `.innerHTML` 속성을 활용해 배열에 담긴 객체 데이터를 순회하며 HTML 텍스트 노드로 생성했습니다.
- **이벤트 핸들링:** 버튼의 `onclick` 속성과 텍스트 박스의 `onkeydown` 이벤트를 사용하여 사용자의 입력을 감지했습니다.
- **데이터 직렬화 및 저장:** `JSON.stringify()`를 통해 객체 배열을 문자열로 변환하여 스토리지에 저장하고, 꺼낼 때는 `JSON.parse()`로 복원했습니다.
- **배열 제어:** `.push()` 메서드로 새 객체를 추가하고, `.splice()` 메서드로 인덱스를 추적하여 데이터를 잘라냈습니다
